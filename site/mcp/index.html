● ArcGIS Online ● ArcGIS Enterprise Model Context Protocol

An MCP server that speaks fluent ArcGIS.

MCP GIS Hub exposes ArcGIS Online and ArcGIS Enterprise REST APIs as a small set of Model Context Protocol tools — findLayer and queryFeatures — so Microsoft Copilot Studio (or any MCP client) can search, discover, and query live feature layers in natural language.
